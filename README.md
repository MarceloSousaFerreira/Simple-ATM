# Simple Java ATM Simulator 🏦

A console-based Automated Teller Machine (ATM) simulator developed as a beginner-friendly Java project. This application allows users to perform basic banking operations through an interactive terminal menu.

## 🚀 Features

- **Check Balance:** View the current account balance.
- **Make Deposit:** Add money to the account, with validation to prevent negative deposits.
- **Withdraw Money:** Take money out of the account, featuring dual validation:
  - Prevents withdrawing negative amounts.
  - Prevents withdrawing more money than the available balance (Insufficient Funds protection).
- **Interactive Menu:** A `do-while` loop ensures the menu keeps running until the user explicitly chooses to exit.

## 💻 Technologies Used

- **Language:** Java (Java SE)
- **Core Concepts:** - `Scanner` class for user input.
  - `Switch-Case` statements for menu navigation.
  - `If/Else` statements for input validation and business logic.
  - `Do-While` loops for continuous execution.
